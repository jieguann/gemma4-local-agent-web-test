Data Flows

1. Comedy Set Loop

  1. startSet() begins the continuous loop in main.js
  2. If no prior conversation, calls comedyAgent.opener()
  3. Main loop: waitBetweenBits() (3s pause) then runOneBit()
  4. nextBit() checks for pending audience input
  5. If input exists: inferComedyMode() routes to interactive riff or full plan+render pipeline
  6. If no input: autonomous freestyle with optional web search
  7. Tokens stream to TTS and UI simultaneously

2. Text + Voice Sync

  1. runOneBit() creates a streamSpeaker from the TTS synthesizer
  2. Each onToken callback calls streamer.feed(fullText)
  3. The streamer splits text at sentence boundaries and queues each to speechSynthesis
  4. Each utterance's start event fires onReveal(visibleText)
  5. The UI updates the displayed text only when the sentence begins playing
  6. After generation completes, streamer.flush() speaks the remaining text

3. Judge Evaluation

  1. After the comedy bit finishes, the inference engine gets a 650ms cooldown
  2. feedbackAgent.evaluateBit() sends the joke + audience signals + conversation to the judge prompt
  3. If the engine is still busy, retries once after 1300ms
  4. Judge outputs: Score, Emotion, Emojis, Verdict, Reaction, ShouldHeckle, Heckle, Advice
  5. applyCrowdEvaluation() triggers the judge's specific emojis as floating bursts
  6. Updates mood bar, critic panel, and pushes audience signals for next bit

4. Heckle Recovery

  1. If judge score ≤ 45 and shouldHeckle: true, the heckle flow triggers
  2. Heckle is shown as a "crowd" message in the chat
  3. After another inference cooldown, comedyAgent.defendAgainstHeckle() fires
  4. Uses COMEDY_DEFENSE_PROMPT to generate a quick comeback
  5. Defense bit streams with TTS like a normal bit (but skips judge evaluation to avoid infinite loop)

5. Autonomous Tool Use

  1. Before each planned bit, autonomousToolStep() runs COMEDY_TOOL_DECISION_PROMPT
  2. Model decides: output ACTION: web_search({...}) or SKIP
  3. If searching, the web_search tool proxies through Vite to DuckDuckGo / Wikipedia
  4. Search results (top 3, truncated to 160 chars each) feed into the joke prompt as "Facts"
